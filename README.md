# 发票文件批量重命名工具
## 项目介绍
一款**纯前端本地运行、极简UI、全设备自适应**的发票文件批量处理网页工具。
所有文件、Excel、发票资料**全程在浏览器本地运算**，不会上传任何数据至服务器，隐私安全性极高，可在GitHub Pages 长期在线使用。

## 核心功能
### 基础功能（默认普通模式）
1. 支持 **点击上传 + 拖拽上传** Excel对照表、ZIP压缩包
2. 支持外部程序 **API调用上传文件**，可作为后端服务调用
3. 根据Excel编号自动批量重命名文件
4. 命名格式：`物品简写_金额_分类_发票编号`
5. 字段为空自动填充 `无`，避免文件名异常
6. 完整运行日志：
   - 重命名成功数量、失败数量
   - 失败文件明细与失败原因
   - Excel存在编号、但无对应文件清单
7. 文件上传成功后拖拽框自动**绿色高亮提示**
8. 处理完成按钮自动变为 `重新批量重命名`，支持一键重置
9. 自动导出重命名完成后的压缩包

### 高级模式（开关开启）
1. 一键将所有文件**统一转为图片**排版整理
2. 文件格式兼容：
   - 图片格式：jpg / png / webp 直接使用
   - PDF文档：自动转图片，**最大只转换前21页**，防止文件过大溢出
   - Word文档：docx/doc 自动渲染转图片，限制前21页
3. 所有图片 **严格按照重命名后的文件名称中文顺序排序**
4. 自动排版、自适应图片大小，一键导出规整 Word 文档

## 数据安全说明
1. **所有数据100%本地浏览器处理**，不上传、不云端存储、不泄露
2. GitHub Pages 仅托管网页代码，**永远无法获取你的任何发票、Excel业务数据**
3. 全站 HTTPS 加密访问，传输安全
4. 关闭页面、刷新页面所有文件数据自动销毁
5. 无后台、无数据库、无信息采集、无隐私泄露风险
6. 适合企业内部财务、票据隐私文件使用

## 部署方式
1. 在 GitHub 新建公开/私有仓库
2. 将项目 `index.html` 上传至仓库根目录
3. 开启仓库 `Pages` 服务
4. 自动生成在线访问链接，任意设备打开即可使用

## API 调用说明
可被其他程序、脚本、系统远程调用上传文件
```js
// 上传Excel对照表
window.uploadExcel(file)

// 上传发票压缩包
window.uploadZip(file)
```

## 使用流程
1. 上传 Excel 对照表（必须包含：编号、物品简写、金额、分类、发票编号）
2. 上传包含票据文件的ZIP压缩包
3. 点击 **开始批量重命名** 自动处理
4. 查看日志统计，下载重命名压缩包
5. 如需整理文档：打开顶部「高级模式」→ 一键导出排版Word

## 环境依赖
全部CDN在线引入，无需本地安装依赖
- xlsx Excel解析
- jszip 压缩包解析
- pdfjs PDF转图片
- docx-preview Word渲染
- docx Word文档生成
- html2canvas 文档转图

## 注意事项
1. 压缩包内文件必须**以数字编号命名**，如 `1.pdf` `2.jpg`
2. Excel编号需要和文件编号一一对应
3. PDF、Word页数较多时自动截断前21页，避免Word文件臃肿
4. 仅支持zip格式压缩包上传
5. 所有操作离线可用，联网仅加载前端依赖库
